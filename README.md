# 支付宝支付沙箱测试

这是一个简单的支付宝支付沙箱测试页面，用于测试支付宝JSAPI支付功能。

## 功能特性

- 支付宝JSAPI支付调用
- 环境检测（确保在支付宝环境中运行）
- 错误处理和用户友好提示

## 使用方法

1. 在支付宝App中打开 `pay.html` 页面
2. 点击"发起支付"按钮
3. 完成支付测试流程

## 部署说明

### 微信云服务托管

1. 将项目推送到GitHub仓库
2. 在微信云服务控制台创建静态网站托管服务
3. 绑定GitHub仓库并部署

### 本地开发

```bash
# 安装依赖
npm install

# 启动本地服务器
npm start
```

## 注意事项

- 此页面必须在支付宝App中打开才能正常工作
- 确保使用有效的沙箱测试交易号
- 生产环境请使用真实的交易号和安全配置

## 技术栈

- HTML5
- JavaScript (ES6)
- 支付宝JSAPI SDK
